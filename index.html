The Pixel Size determines the size of the squares which your image is divided into, for instance a 1000x1000 image with Pixel Size 5 would result in 200 * 200 = 40000 blocks

The Colour Threshold determines how similar colors have to be to be in the same block. 0 = no difference, <1 = imperceptible, 2-10 = quite similar

The Block Size determines the size of the blocks in the Skap Map, in Skap units
Skap Pixel Art Generator was made by NKY, for Skap mapmakers.
